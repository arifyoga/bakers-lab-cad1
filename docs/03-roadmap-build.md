```
# Roadmap Build - Baker's Lab
```

> Prioritas: Mulai dari yang PALING MUDAH dan PALING BERMANFAAT untuk saya sendiri.

## 🏗️ Fase 0: Persiapan (Sekarang - 1 Minggu)

- [ ] Bisa buka repository di GitHub
- [ ] Bisa commit dan push perubahan
- [ ] Paham struktur HTML dasar
- [ ] Bisa menulis Markdown (saya sedang belajar sekarang)

---

## 🏗️ Fase 1: Katalog Resep Statis (Minggu 1-2)

**Goal:** Bisa menampilkan daftar resep di halaman web (hardcode dulu, belum pakai database)

**Sub-tugas:**

- [ ] Buat halaman `index.html` dengan daftar 5 resep contoh
- [ ] Tambahkan CSS sederhana agar rapi
- [ ] Tampilkan gambar (placeholder dulu)
- [ ] Buat halaman detail resep (klik nama resep → lihat detail)

**Teknologi:** HTML + CSS + JavaScript dasar

**Ukuran keberhasilan:** Saya bisa melihat semua resep favorit saya di satu halaman web.

---

## 🏗️ Fase 2: CRUD Resep (Minggu 3-4)

**Goal:** Bisa menambah, mengedit, menghapus resep (pakai `localStorage` dulu, belum backend)

**Sub-tugas:**

- [ ] Buat form tambah resep (nama, bahan, langkah, foto)
- [ ] Simpan ke localStorage
- [ ] Tampilkan daftar dari localStorage
- [ ] Fitur edit dan hapus

**Teknologi:** HTML + CSS + JavaScript (DOM + localStorage)

**Ukuran keberhasilan:** Saya bisa menyimpan resep baru dan tidak hilang setelah refresh browser.

---

## 🏗️ Fase 3: Fitur Analisis Resep (Minggu 5-6)

**Goal:** Dari resep yang tersimpan, hitung baker's percentage dan tampilkan insight

**Sub-tugas:**

- [ ] Tambahkan field "berat tepung" di setiap resep
- [ ] Hitung rasio bahan lain terhadap tepung
- [ ] Tampilkan visual sederhana (progress bar, warna)

**Ukuran keberhasilan:** Saya bisa lihat resep mana yang "terlalu manis" atau "terlalu basah" dari angkanya.

---

## 🏗️ Fase 4: Modul UMKM (Minggu 7+)

**Goal:** Tambahkan kalkulator kapasitas produksi + harga jual

**Sub-tugas:**

- [ ] Buat input form untuk kapasitas terpasang
- [ ] Tambahkan field downtime dan setup time
- [ ] Hitung kapasitas efektif
- [ ] Kalkulator harga produksi

**Ukuran keberhasilan:** Saya bisa memberikan rekomendasi harga jual ke teman yang punya bakery.

---

## ⚠️ Catatan Penting

1. **Prioritas bisa berubah** tergantung apa yang saya pelajari dan butuhkan.
2. **Jangan pindah ke fase berikutnya** sebelum fase sebelumnya benar-benar selesai dan saya paham kodenya.
3. **Setiap selesai satu subtugas, commit ke GitHub** dengan pesan yang jelas.

---

## 🎯 Target Akhir 2026

Saya ingin punya aplikasi web yang:

- Bisa dipakai sendiri untuk mengelola resep
- Bisa diakses oleh teman/keluarga (dideploy ke hosting)
- Menjadi portofolio yang menunjukkan saya bisa programming
