# 🥐 Baker's Lab

> _Laboratorium Digital untuk Eksperimen dan Manajemen Produksi Bakery_

---

## 🎯 Visi Proyek

Membangun aplikasi web yang membantu:

- **Pembuat roti rumahan** menyimpan dan menganalisis resep
- **UMKM bakery** menghitung kapasitas produksi dan harga jual
- **Eksperimenter** mencatat percobaan resep secara terstruktur
